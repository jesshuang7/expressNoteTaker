## About the Project:
The focus of this project was to create an application that can be used to write, save, and delete notes. This application will use an express backend and save and retrieve note data from a JSON file.


![Project Gif](Assets/expressnotetaker.gif)

### User Story

```
AS A user, I want to be able to write and save notes

I WANT to be able to delete notes I've written before

SO THAT I can organize my thoughts and keep track of tasks I need to complete
```

### Built With:
* This app is build upon [Inquirer npm package](https://github.com/SBoudrias/Inquirer.js/), [express](https://expressjs.com/en/5x/api.html), and [inquirer](https://www.npmjs.com/package/inquirer).
